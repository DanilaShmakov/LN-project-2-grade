Обзор Eclair Mobile

Обзор кошелька Eclair Mobile

  • Установка кошелька

    Установки кошелька «Eclair Mobile доступно для мобильных телефонов на базе Андроид. Кошелек бесплатен для скачивания в официальном магазине «GooglePlay». Кошелек легко находится после ввода в строке поиска «Eclair». Иконка представляет собой трехцветную молнию: тёмно-синяя, синяя и фиолетовая. Размер программы после установки 26 МБ.

  • Начало работы

    При первом запуске кошелька, предлагается создать новый или восстановить старый кошелек (рис. 1). После входа в кошелек, происходит переход на главное окно, в котором есть 3 вкладки: «Получить» (рис. 2), «Транзакции» (рис. 3), «Каналы» (рис. 4).

    При следующих запусках кошелька, будет запрошен шестизначный PIN, который был установлен при создании кошелька (рис. 5).

  • Меню

    В правом верхнем углу экрана расположена кнопка «Еще» , по нажатии на которую открывается выпадающее меню (рис. 6) с кпопками "Network Info"(«Информация о сети»), "Settings"(«Настройки»), "About/Help"(«Помощь»), по нажатии на которые открываются соответсвующие окна.

    В окне "Network Info"(«Информация о сети») содержатся сведения о вашей Bitcoin сети, Lightning сети и ключ, позволяющий генерировать адреса. (рис. 7)
    Окно "Settings"(«Настройки») предоставляет возможность изменить URL проводника, вкл/выкл тактильную обратную связь при вводе PIN-а, выбрать отображаемую валюту и криптовалюту, количество отображаемых цифр после запятой, вкл/выкл защиту Lightning транзакций и вкл/выкл возможность Lightning транзакций. (рис. 8)
    В окне "About/Help"(«Помощь») содержатся сведения о кошельке и полезные ссылки github репозиторий. (рис. 9)

  • Переводы

    Для осуществления перевода необходимо иметь средства на счету.(рис.10) После того, как средства у нас появились, подключаемся к ноде и создаем с ней канал.(рис.11) Создание канала необходимо только для переводов LN, для обычных переводов требуется в разделе транзакций нажать на синию кнопку со стрелкой. Минимальная сумма для создания канала - 100000 сат. Канал открывается от 20 минут до 12 часов, его статус готовности будет указан в окне.(рис.12). После начала открытия деньги поступают на канал.(рис.13) Можно использовать канал после того, как его статус станет NORMAL (рис.14).
